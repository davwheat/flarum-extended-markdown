# Extended Markdown

A proof-of-concept to add additional BBCode to Flarum, meeting asks of the giffgaff community
### Installation

```sh
composer require imorland/flarum-extended-markdown
```

### Updating

```sh
composer update imorland/flarum-extended-markdown
```
